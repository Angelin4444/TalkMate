# TalkMate

TalkMate is a smart conversational chatbot focused on emotional support, user interaction, and accessible mental wellness assistance.

## Disclaimer

This project was created for academic and educational purposes only.

TalkMate is a prototype project and not a fully functional commercial application. Some features, such as the Contact Us page and certain chatbot functionalities, may not work completely or send real messages/emails.

The website was developed to demonstrate UI/UX design, chatbot integration using Dialogflow, and front-end web development using HTML, CSS, and JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript
- Dialogflow

## Project Purpose

This project was developed as part of a university coursework/project submission.
